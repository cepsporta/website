GAMES

Cricket

This event contains 15 members from each group, 11 main players and 4 substitutes. The game will be held as same as the IPL Qualifier Match.

Badminton

1.Singles 2.Doubles
For the each event, there would be only 4 members from each group, 2-Girls, 2-Boys.The game will be a rally of 15-points, with 3 sets each.

Kabaddi

There should be 10 members from each group with 7 as main players and 3 substitutes.

Volley Ball

9 members from each group with 6 as main players and 3 substitutes. Rules and regulations will be announced at the time of event by the Event coordinator.

Premier Futsal

This event is only for girls which contains 3 girls from each group. In this they should score goals as per 3s football regulations.

Dodgeball

This is a girls event with 6 members from each group. Rules and regulations will be announced at the time of event by the Event coordinator.

Basketball

This is a girls event with 3 members from each group. Rules and regulations will be announced at the time of event by the event coordinater.

Marathon

The marathon journey would begin from Poonjar Bus Stand where the flag off will be initiated and will end at College of Engineering, Poonjar on 31/01/2018. The participants would cover a 3.9km hilly track. A cash price awaits the winner.
